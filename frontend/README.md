### Intial setup

```bash
$ git clone https://github.com/KristapsDB/workio
$ cd workio/frontend
$ yarn
```

> Requires a Firebase config file. Create one at `/src/config/firebase.json`.

### Run the development server

```bash
$ yarn start
```

### Create a production build

```bash
$ yarn build
```

### Deploy to firebase hosting

```bash
$ yarn deploy
```
